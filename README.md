# Lab6 Basic Components App

This Expo-based React Native application demonstrates a variety of core components and layout techniques. The code was updated to provide a polished "perfect" sample app.

## Features

- Modular design with custom `Header` and `Footer` components
- `SafeAreaView` and `ScrollView` for proper layout on different devices
- Image display using a remote URI
- Text input with two-way state binding
- Simple counter with increment and reset buttons
- Basic styling and theming with `StyleSheet`

## Running the App

Make sure you have the Expo CLI installed:

```bash
npm install -g expo-cli
```

Then start the project:

```bash
cd Lab6_BasicComponents
npm install
npm run start
```

Choose a platform (iOS, Android, or web) via the Expo developer tools.
